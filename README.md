# Go api sample

Golang api using gin framework.

### Feature

- [x] API Documentation Using Swagger
- [x] And More

## Command

- Install node modules

```go
$ go get . || go mod
```

- Build application

```go
$ go build -o main
```

- Start application in development

```go
$ go run main.go
```
